# nest-graphql-boilerplate

Boilerplate for Nest server with TypeORM and GraphQL.

## Installation
```bash
yarn
```
## Development
### Linting
```bash
yarn run lint
yarn run lint:fix
```

### Running
```bash
yarn run dev
```

### Testing
```bash
yarn run test
yarn run test:watch
yarn run test:coverage
```

## Production
### Building
```bash
yarn run build
```

### Running
```bash
yarn run start
```
